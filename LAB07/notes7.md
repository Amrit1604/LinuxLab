# LAB EXPERIMENT - 7

**Implement chown, chmod command with their options**

***CHOWN-***
```
chown labop2:Newpro text.txt
chown -R labop2:Newpro mydirectory/
chown -R -c labop2:Newpro mydirectory/
```

***CHMOD-***
```
chmod 755 text.txt
chmod -R 755 mydirectory/
chmod -R -c 755 mydirectory/
```
